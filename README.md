# Portfolio

This is a portfolio website built using Next.js. It showcases my projects, skills, and experience in different areas of development.

## Quick Start

To get started with the project, follow these steps:

1. Clone the repository:
2. Install dependencies:

    ```bash
    npm install
    ```

3. Run the development server:

    ```bash
    npm run dev
    ```

4. Open your browser and navigate to `http://localhost:3000` to see the website in action.
